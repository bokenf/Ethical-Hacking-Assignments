
# Assignment 2: Scanning and Enumeration

## Objective
To actively scan a specific target machine within a controlled lab environment to discover open ports, live hosts, and active services.

## Scope of Work
* **Host Discovery:** Identifying live workstations across internal departments (e.g., Management, Finance, HODs).
* **Port Scanning:** Checking for open TCP/UDP ports vulnerable to external network access.
* **Service & OS Detection:** Extracting application version banners and fingerprinting operating system flavors.

## Tools Used
* `Nmap` (Service banner grabbing and OS detection flags)
* `OpenVAS` (Vulnerability scanning)
