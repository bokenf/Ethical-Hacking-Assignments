# Assignment 3: Vulnerability Exploitation & Mitigation

## Objective
To test a web application within a localized, controlled lab environment for critical security vulnerabilities, execute safe exploitation, and implement defensive fixes.

## Areas of Focus
1. **Authentication Testing:** Attempting password cracking and auditing weak credential policies.
2. **Web Security:** Testing input fields for common injections (e.g., PHP database or SQL structural modifications).
3. **Exploitation Frameworks:** Executing payloads against outdated, unpatched system software.
4. **Remediation & Mitigation:** Proposing code-level patches or configurations to shut down the discovered attack vectors.

## Tools Used
* `Metasploit Framework`
* `Burp Suite`
* `John the Ripper` / `Hydra`
